## Getting Started
### Prerequisites

* HTML CSS JS
* Python
* Django


### Installation

* Backend

    ```Python
    pip install -r requirements.txt
    python manage.py makemigrations
    python manage.py migrate
    ```

### Usage

* To Create Super User

    ``` python
    python3 manage.py createsuperuser
    ```
  
* To Runserver

    ``` python
    python3 manage.py runserver
    ```
